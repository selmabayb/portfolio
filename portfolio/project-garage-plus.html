Le Projet

Digitaliser la gestion d’un garage automobile et centraliser les processus métiers.

Les garages utilisent souvent des outils fragmentés. GaragePlus centralise tout : clients, véhicules, devis, factures et rendez-vous pour éviter les erreurs et gagner du temps.

Python Django PostgreSQL Full-Stack
Type : Application web full-stack (Projet Universitaire)
Rôle : Développeur Full-Stack
Code : Voir le repository GitHub
Objectifs Clés
  • Centraliser la données (Clients/Véhicules)
  • Automatiser les documents (Devis/Factures PDF)
  • Gérer l'atelier (Rendez-vous)
  • Sécuriser les accès (Rôles Client/Admin)

Fonctionnalités & Solution

Une réponse complète aux besoins métier d'un garage.

Gestion Clients & Véhicules

CRUD complet permettant de lier chaque client à son ou ses véhicules avec historique d'interventions.

Devis & Factures (PDF)

Génération automatique de documents PDF professionnels prêts à être envoyés. Suivi des statuts.

Rendez-vous Atelier

Module de prise de rendez-vous pour organiser la charge de travail de l'atelier.

Authentification

Système sécurisé avec séparation claire des rôles (Espace Client vs Espace Gestionnaire).
